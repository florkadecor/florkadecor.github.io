BCA ID

Ketentuan BCA ID PT Bank Central Asia Tbk (BCA)

A. DEFINISI

  1. BCA ID adalah kode identifikasi yang diciptakan oleh Nasabah maupun bukan Nasabah yang terdiri dari angka, huruf, dan/atau simbol tertentu untuk mengakses e-Channel BCA ID.
  2. Password BCA ID adalah kata sandi pribadi yang diciptakan dan wajib dimasukkan Pengguna BCA ID untuk dapat menggunakan BCA ID.
  3. Nasabah adalah perorangan yang memiliki rekening dana di BCA dan/atau kartu kredit BCA.
  4. Pengguna BCA ID adalah perorangan baik Nasabah maupun bukan Nasabah yang menggunakan BCA ID.
  5. Email adalah alamat surat menyurat elektronik yang didaftarkan oleh Pengguna BCA ID di BCA sehubungan dengan penggunaan BCA ID.
  6. e-Channel BCA ID adalah kanal elektronik BCA yang dapat diakses dengan menggunakan BCA ID untuk melakukan transaksi perbankan dan mendapatkan informasi terkait layanan atau produk yang dipasarkan melalui BCA, yang akan diberitahukan oleh BCA dari waktu ke waktu dalam bentuk dan melalui sarana apa pun sesuai ketentuan hukum yang berlaku.
  7. PIN BCA ID adalah kode sandi pribadi yang diciptakan Pengguna BCA ID, yang terdiri dari 6 (enam) digit numerik dan digunakan untuk melakukan transaksi-transaksi tertentu sesuai ketentuan yang berlaku di BCA.

B. REGISTRASI BCA ID

  1. Calon Pengguna BCA ID dapat melakukan registrasi BCA ID melalui berbagai e-Channel BCA ID.
  2. Pengguna BCA ID wajib memberikan data yang dibutuhkan BCA untuk melakukan registrasi BCA ID.
  3. Untuk dapat menggunakan BCA ID, Pengguna BCA ID wajib melakukan aktivasi BCA ID dengan mengklik tautan yang dikirimkan oleh BCA ke Email yang telah didaftarkan Pengguna BCA ID dalam jangka waktu yang diberitahukan pada Email
  4. Pengguna BCA ID wajib mendaftarkan Email milik Pengguna BCA ID sendiri. Pengguna BCA ID bertanggung jawab sepenuhnya atas segala akibat yang timbul atas pendaftaran dan penggunaan Email milik pihak lain oleh Pengguna BCA ID dan Pengguna BCA ID dengan ini membebaskan BCA dari segala macam tuntutan, gugatan, dan/atau tindakan hukum lainnya dalam bentuk apa pun dari pihak mana pun sehubungan dengan hal tersebut.
  5. Email yang terdaftar di BCA akan digunakan untuk keperluan pengiriman:
    1. notifikasi atas transaksi yang dilakukan Pengguna BCA ID dengan menggunakan BCA ID;
    2. tautan untuk aktivasi BCA ID saat registrasi dan pembukaan blokir BCA ID; dan
    3. informasi lainnya dari BCA termasuk materi promosi.
  6. Pengguna BCA ID setuju bahwa atas persetujuan Pengguna BCA ID, BCA dapat mengirimkan materi promosi ke Email Pengguna BCA ID yang terdaftar di BCA.

C. BCA ID DAN KETENTUAN TRANSAKSI

  1. Pengguna BCA ID dapat memiliki beberapa BCA ID.
  2. Pengguna BCA ID harus mengaitkan BCA ID dengan minimal 1 (satu) rekening untuk dapat melakukan transaksi finansial maupun non finansial pada e-Channel BCA ID.
  3. Satu rekening hanya dapat dikaitkan dengan 1 (satu) BCA ID. Namun demikian, rekening dengan status rekening gabungan dapat dikaitkan dengan lebih dari 1 (satu) BCA ID.
  4. Satu kartu kredit hanya dapat dikaitkan dengan 1 (satu) BCA ID.
  5. Satu BCA ID hanya dapat dikaitkan dengan 1 (satu) Email dan 1 (satu) Email hanya dapat dikaitkan dengan 1 (satu) BCA ID. Pengguna BCA ID dapat mengubah Email yang dikaitkan dengan BCA ID dengan mengikuti ketentuan yang berlaku di BCA.
  6. Untuk dapat melakukan login pada e-Channel BCA ID, Pengguna BCA ID harus memasukkan BCA ID dan Password BCA ID.
  7. Pengguna BCA ID harus menggunakan PIN BCA ID yang benar setiap kali Pengguna BCA ID melakukan transaksi tertentu yang membutuhkan PIN BCA ID pada e-Channel BCA ID.
  8. Pengguna BCA ID wajib mengikuti ketentuan yang berlaku pada setiap e-Channel BCA ID setiap kali Pengguna BCA ID melakukan transaksi pada e-Channel BCA ID tersebut yang akan diberitahukan oleh BCA kepada Pengguna BCA ID dalam bentuk dan melalui sarana apa pun sesuai ketentuan hukum yang berlaku.

D. PENGAMANAN BCA ID, PASSWORD BCA ID, DAN PIN BCA ID

  1. Pengguna BCA ID wajib menjaga kerahasiaan BCA ID, Password BCA ID, dan PIN BCA ID dengan cara:
    • tidak memberitahukan BCA ID, Password BCA ID, dan PIN BCA ID kepada orang lain termasuk kepada anggota keluarga atau orang terdekat Pengguna BCA ID;
    • tidak menyimpan BCA ID, Password BCA ID, dan PIN BCA ID pada handphone, benda-benda lainnya atau sarana apa pun yang memungkinkan BCA ID, Password BCA ID, dan PIN BCA ID diketahui oleh orang lain;
    • berhati-hati dalam menggunakan BCA ID, Password BCA ID, dan PIN BCA ID agar tidak terlihat oleh orang lain;
    • tidak menggunakan BCA ID, Password BCA ID, dan PIN BCA ID yang ditentukan atau dipilihkan oleh orang lain, atau yang mudah diterka seperti tanggal lahir atau kombinasinya dan nomor telepon.
  2. Menyimpang dari ketentuan butir D.1 di atas, BCA dapat meminta BCA ID dari Pengguna BCA ID untuk keperluan penanganan keluhan yang disampaikan oleh Pengguna BCA ID.
  3. Segala penyalahgunaan:
    1. BCA ID yang bukan disebabkan karena kesalahan BCA; dan
    2. Password BCA ID dan PIN BCA ID;
    merupakan tanggung jawab Pengguna BCA ID sepenuhnya. Pengguna BCA ID dengan ini membebaskan BCA dari segala tuntutan, gugatan, dan/atau tindakan hukum lainnya yang timbul, baik dari pihak lain maupun Pengguna BCA ID sendiri sebagai akibat penyalahgunaan BCA ID, Password BCA ID, dan PIN BCA ID.
  4. Pengguna BCA ID setiap saat dapat mengubah Password BCA ID melalui menu yang disediakan.
  5. Apabila BCA ID, Password BCA ID, dan PIN BCA ID milik Pengguna BCA ID diketahui oleh pihak lain, Pengguna BCA ID harus segera memberitahukan hal tersebut kepada Halo BCA agar dapat dilakukan pemblokiran BCA ID. Segala instruksi transaksi berdasarkan penggunaan BCA ID, Password BCA ID, atau PIN BCA ID yang terjadi sebelum BCA menerima pemberitahuan tersebut dari Pengguna BCA ID menjadi tanggung jawab Pengguna BCA ID sepenuhnya.

E. PEMBLOKIRAN DAN PENGHAPUSAN BCA ID

  1. BCA ID akan diblokir jika Pengguna BCA ID melakukan hal-hal sebagai berikut:
    • Salah memasukkan Password BCA ID sebanyak tiga kali berturut-turut; atau
    • Mengajukan permohonan pemblokiran BCA ID.
  2. Pengguna BCA ID tidak dapat menggunakan BCA ID yang diblokir untuk mengakses seluruh
    e-Channel BCA ID.
  3. Atas pertimbangan tertentu BCA berhak untuk melakukan pemblokiran atau penghapusan BCA ID dan penghapusan koneksi BCA ID dengan Email yang dihubungkan dengan BCA ID yang dihapus tersebut, antara lain jika:
    1. BCA ID tidak aktif dalam jangka waktu yang ditentukan oleh BCA yang akan diberitahukan dalam bentuk apa pun melalui sarana apa pun sesuai ketentuan hukum yang berlaku;
    2. Pengguna BCA ID tidak melewati proses verifikasi data sesuai dengan ketentuan yang berlaku di BCA;
    3. seluruh rekening Pengguna BCA ID di BCA telah tutup;
    4. terdapat indikasi bahwa BCA ID, Password BCA ID, dan/atau PIN BCA ID telah diketahui oleh pihak selain Pengguna BCA ID atau disalahgunakan oleh pihak lain;
    5. terdapat transaksi finansial atau transaksi lainnya yang dilakukan menggunakan BCA ID di luar batas penggunaan wajar atau yang terindikasi fraud:
    6. Pengguna BCA ID tidak memenuhi ketentuan hukum yang berlaku;
    7. Pengguna BCA ID tidak memberikan informasi dan dokumen pendukung sesuai ketentuan hukum yang berlaku;
    8. Pengguna BCA ID diketahui dan/atau patut diduga menggunakan dokumen palsu dan/atau memberikan data yang tidak benar kepada BCA;
    9. Pengguna BCA ID menyampaikan informasi yang diragukan kebenarannya; atau
    10. Pengguna BCA ID memiliki sumber dana transaksi yang diketahui dan/atau patut diduga berasal dari hasil tindak pidana.
  4. Untuk keperluan pembukaan pemblokiran BCA ID, Pengguna BCA ID harus menghubungi Halo BCA dan mengikuti instruksi yang disampaikan oleh petugas Halo BCA.
  5. Pengguna BCA ID tidak dapat melakukan penghapusan BCA ID sendiri dengan alasan apa pun.

F. FORCE MAJEURE

Dalam hal BCA tidak dapat melaksanakan instruksi dari Pengguna BCA ID, baik sebagian maupun seluruhnya karena kejadian-kejadian atau hal-hal di luar kekuasaan atau kemampuan BCA, termasuk namun tidak terbatas pada  bencana alam, perang, huru-hara, peralatan/sistem/transmisi dalam keadaan tidak berfungsi, terjadinya gangguan listrik, gangguan telekomunikasi, adanya kebijakan pemerintah yang melarang BCA memberikan fasilitas terkait BCA ID, serta kejadian-kejadian atau hal-hal lain di luar kekuasaan atau kemampuan BCA, maka Pengguna BCA ID dengan ini membebaskan BCA dari segala macam tuntutan, gugatan, dan/atau tindakan hukum lainnya dalam bentuk apa pun dari pihak mana pun terkait dengan hal tersebut.

G. PENANGANAN KELUHAN

  1. Pengguna BCA ID dapat menghubungi Halo BCA atas setiap keluhan yang berkenaan dengan BCA ID. BCA akan menanggapi keluhan tersebut sesuai dengan ketentuan hukum yang berlaku.
  2. Setiap keluhan terkait BCA ID harus disampaikan oleh Pengguna BCA ID kepada BCA dalam waktu selambat-lambatnya 3 (tiga) bulan sejak tanggal penggunaan BCA ID yang mengakibatkan keluhan terkait. BCA berhak menolak melayani keluhan Pengguna BCA ID yang disampaikan kepada BCA setelah lewatnya jangka waktu tersebut.
  3. Pengguna BCA ID setuju bahwa setiap perselisihan atau perbedaan pendapat yang timbul dari dan/atau berkenaan dengan pelaksanaan Ketentuan BCA ID PT Bank Central Asia Tbk (“Ketentuan”) ini akan diselesaikan dengan cara musyawarah.
  4. Setiap perselisihan atau perbedaan pendapat yang tidak dapat diselesaikan secara musyawarah oleh BCA dan Pengguna BCA ID akan diselesaikan melalui fasilitasi perbankan di Bank Indonesia atau Otoritas Jasa Keuangan atau mediasi yang dilakukan melalui Lembaga Alternatif Penyelesaian Sengketa yang tercantum dalam Daftar Lembaga Alternatif Penyelesaian Sengketa yang ditetapkan oleh Otoritas Jasa Keuangan.
  5. Setiap perselisihan atau perbedaan pendapat yang tidak dapat diselesaikan baik secara musyawarah, fasilitasi perbankan, dan/atau mediasi sebagaimana dimaksud dalam butir G.4 di atas akan diselesaikan melalui Pengadilan Negeri Jakarta Pusat, dengan tidak mengurangi hak BCA untuk mengajukan gugatan atau tuntutan melalui Pengadilan Negeri lainnya dalam wilayah Republik Indonesia.

H. BAHASA

Ketentuan BCA ID BCA ini dibuat dalam versi bahasa Indonesia dan bahasa Inggris. Apabila terdapat perbedaan antara versi bahasa Indonesia dan bahasa Inggris, maka yang berlaku adalah versi bahasa Indonesia.

I. LAIN-LAIN

  1. Pengguna BCA ID wajib segera melaporkan kepada BCA secara tertulis apabila terjadi perubahan data Pengguna BCA ID.
  2. BCA berhak untuk mengubah Ketentuan ini yang akan diberitahukan oleh BCA kepada Pengguna BCA ID dalam bentuk dan melalui sarana apa pun sesuai dengan ketentuan hukum yang berlaku.
  3. Pengguna BCA ID dengan ini menyatakan tunduk pada Ketentuan ini serta seluruh syarat dan ketentuan yang berlaku di BCA yang mengatur mengenai jasa, fasilitas, dan transaksi yang dapat dilakukan dengan menggunakan BCA ID yang akan diberitahukan oleh BCA kepada Pengguna BCA ID dalam bentuk dan melalui sarana apa pun sesuai dengan ketentuan hukum yang berlaku.
