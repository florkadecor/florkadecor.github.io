Lebih Lengkap Tentang Debit BCA Contactless

Cara Aktivasi Debit BCA Contactless

Langkah 1/7

Buka aplikasi BCA mobile

Langkah 2/7

Masukkan "Kode Akses m-BCA"

Langkah 3/7

Klik menu "Akun Saya"

Langkah 4/7

Plih menu "Kontrol"

Langkah 5/7

Nyalakan Toggle ‘Transaksi Debit Contactless’ untuk mengaktifkan atau menonaktifkan fitur contactless

Langkah 6/7

Konfirmasi dengan memasukkan ‘PIN m-BCA’

Langkah 7/7

Kontrol transaksi Debit BCA Mastercard Contactless berhasil
