m-Commerce

Beli voucher prabayar buat pulsa handphone, token listrik, hingga voucher game, bisa langsung lewat fitur m-Commerce di BCA mobile.

Beli Voucher Pulsa dan Google Play
Isi ulang pulsa, paket data, atau Google Play, caranya tinggal:
  • Buka BCA mobile
  • Pilih Menu m-Commerce
  • Pilih Voucher Isi Ulang
  • Pilih Jenis Voucher
Top Up Token Listrik
Beli token listrik sebelum lupa, caranya tinggal:
  • Buka BCA mobile
  • Pilih Menu m-Commerce
  • Pilih antara “PLN Prabayar” atau “PLN Manual Advice”
  • Masukkan No Meter atau ID Pelanggan (dan kode transaksi untuk PLN Manual Advice)
