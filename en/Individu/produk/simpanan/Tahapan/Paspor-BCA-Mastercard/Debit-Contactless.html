BCA Mastercard Debit with Contactless feature

BCA Mastercard Debit with contactless feature can be used Domestically at EDC BCA with the logo or Internationally at all EDCs / payment machines that can accept contactless transactions from cards with the Mastercard logo.

BCA Mastercard Debit with contactless feature can be used Domestically and Internationally for transactions on all compatible EDC/payment machines and for public transportation payment that has RFID (Radio Frequency Identification) technology.

Terms of Debit Contactless

  1. When a customer receives a BCA Mastercard Debit with contactless feature, the contactless feature is not active. Customers must activate the Contactless Feature on BCA mobile and myBCA.
  2. The daily limit for BCA Mastercard Debit with contactless feature without a PIN is Rp1 million/card/day.
  3. There is no additional charge for using the contactless feature on debit card.
  4. BCA Mastercard Debit with contactless feature can be used on BCA EDC machines and abroad on any "Tap Reader" machine that supports the Contactless Feature.
  5. Terms of Debit Contactless transactions
    • Without a PIN: maximum transaction up to IDR 1 million
    • With a PIN: transactions above  IDR 1 million(*)

    (*) this provision applies to transactions in Indonesia. For foreign transactions, refer to the regulations in each country.

How to Make a Debit Contactless Transaction

  1. Make sure the shop/merchant where you transact supports contactless payments
  2. Get your BCA Mastercard Debit with contactless feature ready and make sure the transaction amount is correct
  3. Bring the card closer to the Contactless EDC machine
  4. Transaction completed

Disclaimer:

  • Transactions above IDR 1 million can be made by contactless with a PIN and apply to transactions in Indonesia.
  • Contactless transactions without a PIN abroad shall follow the regulations in each country.
  • Transactions without a PIN and/or signature can still be made abroad.

Security Tips for Transactions using Debit Contactless

  1. Keep your BCA Mastercard Debit with contactless feature safe through the activation settings on BCA mobile and myBCA.
  2. Make sure to store the BCA Mastercard Debit with contactless feature in a safe place.
  3. Make sure that your BCA Mastercard Debit with contactless feature does not change hands.
  4. If you lose your BCA Mastercard Debit with contactless feature, immediately block it through BCA mobile, myBCA or file a report to Halo BCA 1500888 to block the card.

How to control BCA Mastercard Debit Card with Contactless feature via BCA mobile

Activation of the Contactless Debit Feature

  1. Have a BCA Debit Mastercard with a Contactless Feature
  2. Have a financially active BCA mobile account
  3. Go to the “Akun Saya” menu
  4. Select the “Kontrol” menu
  5. Switch on the Debit Contactless transaction
  6. Enter the m-BCA PIN
  7. A confirmation screen appears and selects “OK”
  8. The Debit Contactless feature is automatically activated and ready to use

Set Contactless Debit Without a PIN

  1. The Customer already has a BCA Debit Mastercard with a Contactless Feature
  2. The customer has a financially active BCA mobile account
  3. The customer goes to the “Akun Saya” menu
  4. Select the “Atur Limit” menu
  5. Select Debit Contactless Limit without a PIN
  6. Enter m-BCA PIN to confirm
  7. The daily Debit Contactless Limit without a PIN has been successfully set

How to control BCA Mastercard Debit Card with Contactless feature via myBCA

Activation

  1. Select “Akun Saya”
  2. Click “Kontrol Akun”
  3. Select “Kartu Debit” you want to manage
  4. Click “Kontrol Transaksi”
  5. Switch On on “Transaksi Debit Contactless”
  6. Confirm and enter myBCA PIN
  7. BCA Mastercard Debit Card with Contactless feature is ready to use

Set Limit

  1. Select “Akun Saya”
  2. Click “Kontrol Akun”
  3. Select “Kartu Debit” you want to manage
  4. Click “Atur Limit”
  5. Set BCA Mastercard Debit Card with Contactless feature limit without PIN
  6. Confirm and enter myBCA PIN

Debit Contactless FAQs

What types of BCA Debit Card Mastercard are equipped with contactless feature?

  • BCA Debit cards with Mastercard logo and have a contactless logo or come with logo.

How do I make a transaction using BCA Mastercard Debit with contactless feature?

  • Make sure the store/merchant has accepted contactless payments and that your BCA Debit card supports the contactless feature.
  • Activate the Contactless Debit feature through the “Akun Saya” menu on myBCA or BCA mobile.
  • Prepare the Contactless Debit and make sure the payment amount is correct.
  • Hold the card up to the contactless payment machine.
  • The transaction is complete and keep your Contactless Debit card in a safe place.

How do I activate the BCA Mastercard Debit with contactless feature?

Activate the BCA Mastercard Debit with contactless feature on myBCA:

  • Open and log in to myBCA app
  • Click the “Akun Saya” menu
  • Select “Kontrol Akun”
  • Select “Kartu Debit”
  • Select the Debit Card you want to set up
  • Select “Kontrol Transaksi”
  • Switch on the “Transaksi Debit Contactless” toggle
  • Confirm by entering your myBCA PIN

Activate the BCA Mastercard Debit with contactless feature on BCA mobile:

  • Open BCA mobile,
  • Enter your m-BCA access code
  • Click on “Akun Saya” menu
  • Select the “Kontrol” menu
  • Switch on the “Transaksi Debit Contactless” toggle
  • Confirm by entering your m-BCA PIN

For contactless Debit transactions on public transportation*) abroad, you can do it without activating the contactless feature.

*) Such as Mass Rapid Transit (MRT) in Singapore, Express Rail Linkin Malaysia, The Bangkok Mass Transit System, Public Transportation in Australia (Trains, Trams, Buses, Translink, Transperth), GVB Amsterdam (Metros, Buses, Trams, Ferries), Mass Transit Railway (MTR) in Hongkong, Public Transportation in the United Kingdom (Bus, Translink, Metrolink), Riyadh Bus in Saudi Arabia, Public Transportation in Turkey (Bodrum Express Lines, Burulas, Havaist), etc.

Where can customers make BCA Mastercard Debit with contactless feature transactions?

  • Domestic on BCA EDCs
  • Overseas on any EDC/payment machine that accepts contactless transactions from cards with the Mastercard logo.

What is the mechanism for BCA Mastercard Debit with contactless feature transactions abroad?

Contactless Debit transactions abroad can be done at merchants with compatible EDCs/payment machines and public transportation with RFID technology. The Contactless Debit transaction mechanism is generally identical to that in the country.

EDC Transaction:

  • Make sure the store /merchant where you are making transactions supports contactless payments
  • Prepare your Contactless Debit card and make sure the purchase amount is correct
  • Hold the card up to the contactless EDC machine
  • Transaction is complete

Transportation:

  • Hold the card close to the tap machine when entering and exiting transportation.

Each public transportation company applies different policies for accumulating travel expenses and providing information to customers. Therefore, customers are advised to use the application from each public transportation company/provider to see detailed cost information on each trip made.

What is the daily limit of BCA Mastercard Debit with contactless feature?

The daily limit of Contactless Debit without PIN is Rp1 million/card/day.

Can I set the daily limit for BCA Mastercard Debit with contactless feature transactions?

You can set your Contactless Debit transaction limit via myBCA or BCA mobile.

Set your Contactless Debit transaction limit on myBCA:

  • Open and log in to myBCA app
  • Click the “Akun Saya” menu
  • Select the “Kontrol Akun” menu
  • Select “Kartu Debit”
  • Select the Debit card you want to set
  • Select “Atur Limit”
  • Adjust the “Contactless Debit Limit without PIN” as needed
  • Confirm by entering your myBCA PIN

Set your contactless debit transaction limit on BCA mobile:

  • Open BCA mobile
  • Enter your m-BCA access code
  • Click the “Akun Saya” menu
  • Select the “Atur Limit” menu
  • Adjust the “Contactless Debit Limit without PIN” as needed
  • Confirm by entering your m-BCA PIN
  • Done.

What if I lost my BCA Mastercard Debit with contactless feature?

If your Contactless Debit card is lost, immediately block it via myBCA/BCA mobile or report it to Halo BCA 1500888 for blocking.

How to Bock Contactless Debit on myBCA

  • Open and log in to myBCA app
  • Click the “Akun Saya” menu
  • Select the “Kontrol Akun” menu
  • Select “Kartu Debit”
  • Select the Debit card you want to set
  • Select “Blokir ”,
  • Confirm by entering your myBCA PIN

How to Bock Contactless Debit on BCA mobile

  • Open BCA mobile
  • Enter your m-BCA access code
  • Click the “Akun Saya” menu
  • Select the “Blokir” menu
  • Confirm by entering your m-BCA PIN
